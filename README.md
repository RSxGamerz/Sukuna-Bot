- Deploy bot to the Replit.</br>
- Make `env.example` variables Secrets in Replit.</br>
- Give an Random `SESSION_ID` ( Like "dlfhdsfhdsfh" or "fkdsfdhslfhklfd" ) and put it in a `Secrets`.</br>
- Run and Wait until you get the `QR`.</br>
